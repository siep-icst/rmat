# RMAT Generator

Create graph with 100 nodes and 400 edges<br>
print all vertices<br>
print all edges





## Example

    ./rmat_gen.exe 100 400 59 19 19 5

parameters:<br>
v_num, e_num, A, B, C, D<br>
A, B, C, D is 100 times of a, b, c, d<br>
a+b+c+d=1<br>
undirected graph: b=c
